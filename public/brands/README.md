# Logos de marcas

Colocá los logos de las marcas aquí con el nombre que guardaste en la base de datos en el campo `logo_path`.

Ejemplo:
- `amt.png` → logo_path = "amt.png"
- `darkglass.png` → logo_path = "darkglass.png"

Estos archivos son accedidos desde la URL `/brands/nombre.png` en el frontend.
